saya ingin menambahkan fitur pembuatan RAB untuk kegiatan BOK Puskesmas pada aplikasi ini.
saya sudah punya file template RAB Puskesmas Saya dalam bentuk excel. nama templatenya adalah templaterab.xlsx
jadi di dalam pembuatan RAB ini terdiri dari

1. Komponen = {Contoh Peningkatan Layanan Kesehatan Sesuai Siklus Hidup}
2. Rincian Menu = {Contoh Pelaksanaan Kelas ibu Hamil dan Kelas Ibu Balita}
3. Kegiatan = {Contoh Pelaksanaan Kelas Ibu Balita}

setiap kegiatan terdiri dari item-item.
item-item berikut ini bisa saja ada pada tiap kegiatan bisa juga tidak. tergantung kegiatan.

1. Transport Darat = orang/hari/desa/berapa kali. contoh = {2 orang} x {1 hari} x {2 desa} x {12 Kali Kegiatan} x {harga satuan 70.000} = {sub total 3.360.000}
2. Transport Harian = orang/hari/desa/berapa kali. contoh = {2 orang} x {1 hari} x {3 desa} x {12 Kali Kegiatan} x {harga satuan 70.000} = {sub total 5.040.000}
3. Uang Harian = orang/hari/desa/berapa kali. contoh = {2 orang} x {1 har} x {3 desa} x {12 kali kegiatan} x {harga satuan 150.000} = {sub total 10.800.000}
4. snack (opsional tergantung kegiatan. karena ada kegiatan yang menggunakan snack, dan tidak) = contoh {15 dos} x {5 desa} x {12 kali kegiatan} x {harga satuan 24.000} = {sub total 21.600.000}
5. Penggandaan (opsional tergantug kegiatan. karena ada kegiatan yang menggandakan atk dan lain sebagainya.) = contoh {24 lembar} x {3 desa} x {3 kali kegiatan} x {harga per lembar 750} = {sub total 162.000}
6. Transport Peserta (opsional tergantung kegiatan. contohnya transport kader dari desa lokal maupun desa seberang. harga untuk transport desa lokal adalah 70.000 dan desa seberang adalah 150.000)
7. Konsumsi (opsional tergantung kegiatan. hampir sama dengan Snack. tetapi harga untuk konsumsi adalah 59.000).
8. Pembelian Bahan Makanan (opsional tergantung kegiatan) contoh = {1 paket} x {2 kegiatan} x {750.000} = {1.500.000}
10. Jumlah Total adalah jumlah total dari seluruh item yang terdapat kegiatan tersebut.

apakah kita bisa buat tiap item ini dinamis dapat ditambahkan sesuai kebutuhan.
